# crudly
Crudly is an AI-powered CRUD and system generator built with Laravel, Inertia.js, and React. It allows you to generate complete application systems using simple prompts and download ready-to-run source code instantly.

Instead of manually creating repetitive CRUD modules, Crudly lets you describe what you want (for example, Project Management System), and it automatically generates all required modules, pages, and relationships.
